# Tweeter Project

- Tweeter is a simple, single-page Twitter clone.
- HTML, CSS, JS, jQuery and AJAX was used on the front end
- Node, Express and MongoDB was used on the back end


## Final Product

#### URL index page

- Shows list of all of user's short URLs
